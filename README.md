# Text2App
Create a functional Android app from a given text description of the app. This model generates a MIT app inventor export file that should be uploaded to MIT app inventor website to build the app. The current prototype supports 6 functionalities: detect button press, playing video, playing music, detecting acceleration,Â opening camera, text2speech.
